AppSidebar.vue

Navy #0f172a, 240px mặc định → 64px icon-only. Nav groups có section label. Footer user info. Dùng NuxtLink + permission guard giữ nguyên.

AppTopbar.vue

Breadcrumb động từ layoutStore. Notification bell giữ nguyên. User button dùng shadcn DropdownMenu: Profile / Đổi mật khẩu / Đăng xuất.

home-admin.vue (page)
